# 🚀 SiteBuilder - سیستم مدیریت محتوای حرفه‌ای

یک سیستم مدیریت محتوای کامل شبیه وردپرس که با HTML، CSS و JavaScript خالص ساخته شده است.

## ✨ امکانات
- 📊 پیشخوان مدیریت با آمار
- 📝 مدیریت مطالب (ایجاد، ویرایش، حذف)
- 📄 مدیریت صفحات
- 🏷️ مدیریت دسته‌بندی‌ها
- 👤 مدیریت کاربران
- 🖼️ کتابخانه رسانه
- 🧩 سازنده صفحه (Drag & Drop)
- ⚙️ تنظیمات کلی

## 🔑 اطلاعات ورود
- **نام کاربری:** admin
- **رمز عبور:** admin123

## 📂 ساختار پوشه‌ها

sitebuilder/
├── index.html (صفحه اصلی)
├── assets/
│ ├── css/ (استایل‌ها)
│ └── js/ (اسکریپت‌ها)
├── admin/ (پنل مدیریت)
└── uploads/ (فایل‌های آپلود شده)



## 🚀 راه‌اندازی
1. فایل `index.html` را در مرورگر باز کنید
2. روی "ورود به پنل" کلیک کنید
3. با اطلاعات بالا وارد شوید

## 💾 ذخیره‌سازی داده‌ها
همه داده‌ها در `localStorage` مرورگر ذخیره می‌شوند.
